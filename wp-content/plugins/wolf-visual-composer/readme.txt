== Installation ==

1. Upload the `wolf-visual-composer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Being a free product, this plugin is distributed as-is without official support.
Verified customers however, who have purchased a premium theme on [ThemeForest](http://themeforest.net/user/wolf-themes/portfolio?ref=wolf-themes) will have access to the [support forums](http://wlfthm.es/help).
